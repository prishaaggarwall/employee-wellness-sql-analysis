# Employee Wellness SQL Analysis

## 📌 Objective

To analyze employee mental health and workplace conditions using SQL.

## 🛠 Tools Used

* SQL (MySQL/PostgreSQL)
* Dataset: Employee Wellness Survey

## 📊 Analysis Performed

* Descriptive Analysis (distribution, percentages)
* Exploratory Data Analysis (relationships & patterns)

## 🔍 Key Insights

* Employees with benefits report lower work interference
* Younger professionals experience higher stress levels
* Family history increases likelihood of seeking treatment

## 📁 Files Included

* descriptive_analysis.sql
* exploratory_data_analysis.sql

## 📈 Conclusion

Workplace policies, benefits, and culture significantly influence employee mental health.
